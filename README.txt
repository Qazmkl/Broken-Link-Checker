* Notes
****************************************************/
- Runs on JRE version 1.8+ (MUST-HAVE)
- Maven project made in Eclipse
- Needs an internet connection
- Uses JSoup to parse document
- Uses threads (runnables)
- Creates new folders and lets user choose where to save results
- Outputs to Excel file
- Uses a UI
- Can stop mid-scan
- Takes screenshots using Selenium


/****************************************************
* To RUN
****************************************************/
Run JAR file

OR

In Eclipse:
- Open project through Eclipse
- Press F11
